# 🚀 Road Warrior

Road Warrior is a multilingual rider registration and referral platform designed to onboard delivery riders, generate EV and insurance leads, and build a referral-driven rider community.

## 🌐 Live Demo

🔗 https://road-warrior-cyan.vercel.app/

---

## ✨ Features

### 📝 Rider Registration

* Multi-step registration form
* Mobile number verification using OTP
* Form validation and error handling

### 🌍 Multilingual Support

* English
* Hindi (हिंदी)
* Kannada (ಕನ್ನಡ)

### 🔐 Security Features

* OTP verification
* Google reCAPTCHA integration
* Rate limiting to prevent spam submissions
* Duplicate registration prevention

### 👥 Referral System

* Unique referral code generation
* Referral tracking
* Points-based rewards system
* Referral leaderboard

### 📊 Analytics & Lead Generation

* EV interest tracking
* Insurance lead tracking
* Rider demographic insights
* Platform-wise rider analysis

### 🏆 Rider Engagement

* My Score page
* Referral points tracking
* Leaderboard rankings

### 📱 Mobile-Friendly Design

* Responsive UI
* Optimized for desktop and mobile devices

## 🚀 Run Locally

### Clone Repository

```bash
git clone https://github.com/VarshaNayak18/road-warrior.git
cd road-warrior
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_RECAPTCHA_SITE_KEY=your_recaptcha_site_key
```

### Start Development Server

```bash
npm run dev
```
